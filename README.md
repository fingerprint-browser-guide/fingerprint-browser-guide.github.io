# ZkingWorkgo.github.io

ZkingWorkgo 指纹浏览器及配套工具中文研究索引的 GitHub Pages 源文件。

## 包含内容

- `index.html`：网站首页、SEO元数据与结构化数据
- `styles.css`：响应式视觉样式
- `robots.txt`：搜索引擎抓取规则
- `sitemap.xml`：站点地图
- `llms.txt`：面向机器读取的项目说明与入口
- `site.webmanifest`：站点清单
- `404.html`：错误页面
- `assets/`：图标与社交分享图

网站发布地址：<https://zkingworkgo.github.io/>
